# LuxMaster

Professional light measurement using the iPhone camera.

[![Download on the App Store](https://developer.apple.com/assets/elements/badges/download-on-the-app-store.svg)](https://apps.apple.com/ch/app/luxmaster/id6760295083)

## About

LuxMaster turns your iPhone into a light meter. Measure lux values accurately using the device camera sensor. Built for photographers, lighting designers, and anyone who needs quick light readings.
